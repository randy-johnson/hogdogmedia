# Hog Dog Media — Design System

A media-brand design system built around the duality of **hog and dog**: two voices, one network. Cream paper, deep ink, a single coral accent — soft, warm, broadcast-worthy.

## Voice & content fundamentals

Hog Dog Media is **two voices in conversation**. Everything you write should feel like a dialogue — not a monologue. Use opposing pairs (loud/quiet, fast/slow, builder/host) to mirror the hog/dog duality.

- **Tone:** confident, warm, occasionally irreverent. Never corporate, never cringe.
- **Punctuation:** em-dashes welcome. Semicolons are fine.
- **Headlines:** short and declarative. ALL CAPS is reserved for Anton display text.
- **Body copy:** sentence case. Never title-case mid-sentence.
- **Numbers:** spelled out under ten ("two hosts"); numerals for stats ("184K subscribers").

## Visual foundations

**The mascot is sacred.** Always present the full circular mascot — never crop, recolor, or redraw the hog or dog separately, except for the cropped flank treatment in the horizontal lockup (V4) which is officially deprecated.

**Cream is the default canvas.** Dark ink-warm surfaces are reserved for video bumpers, social drop cards, and dark-mode product UI.

**Coral is a single drop.** One coral element per composition is enough. If you need more emphasis, increase weight or scale, not chroma.

**Negative space is half the design.** Everything in the brand breathes — mimic the cream halo around the mascot itself.

## Iconography

We do not have a custom icon set. Use **Lucide** (or any minimal stroke-based set) at 1.5px stroke weight, ink color, never coral. Icons are functional, not decorative.
